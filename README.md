# Vrinda Store Data Analysis Project
This repository contain an Excel-Based Data analysis project for **Vrinda Store** to analyze sales,customer demogrphic and order trends.
## Key Insights & Observation
### 1. Sales performance
* **Top 3 channels.** 
1.Amazon(35.5%)
2.Flikart(21.6%)
3.Meentra(23.3%)
* **Top 3 Location.**
1. TAMIL NADU
2. TELENGANA
3. UTTAR PRADESH
### 2. CUSTOMER DEMOGRAPHIC
* **Targeted Customer** The majority of buyer are **Women** Contributing to aproximately **64%** of total sales. 
* ***Age Group** The most active purchasing age bracket is **Adult(20-50)**.
### 3. Product & Order Analytics
* **Best Selling Category**
* [Blouse,Kurta,Saree]
* **Order Status.**
**[92%]** of Order delivered , while [3%] Faced canceled
